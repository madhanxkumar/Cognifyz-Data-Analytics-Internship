# Cognifyz Data Analytics Internship Project

## Overview

This repository contains the tasks completed during the Cognifyz Technologies Data Analytics Internship. The project focuses on analyzing restaurant data using Python, Pandas, Matplotlib, and Jupyter Notebook to extract meaningful insights and visualize business trends.

## Tools and Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Anaconda

## Dataset Description

The dataset contains restaurant information including:

* Restaurant Name
* City
* Cuisines
* Aggregate Rating
* Votes
* Price Range
* Online Delivery Availability
* Latitude and Longitude
* Table Booking Information

---

# Level 1 Tasks

## Task 1: Top Cuisines Analysis

* Identified the top 3 most common cuisines.
* Calculated percentage distribution of cuisines.
* Created visualizations for cuisine popularity.

### Key Finding

North Indian cuisine was the most common cuisine in the dataset.

---

## Task 2: City Analysis

* Identified cities with the highest restaurant count.
* Calculated average ratings for each city.
* Determined the highest-rated city.

### Key Finding

New Delhi had the highest number of restaurants in the dataset.

---

## Task 3: Price Range Distribution

* Analyzed restaurant distribution across price categories.
* Calculated percentage contribution of each category.
* Created bar chart visualizations.

### Key Finding

Price Range 1 was the most common category.

---

## Task 4: Online Delivery Analysis

* Calculated online delivery availability percentages.
* Compared ratings of restaurants with and without online delivery.

### Key Finding

Restaurants offering online delivery had higher average ratings.

---

# Level 2 Tasks

## Task 1: Restaurant Ratings Analysis

* Analyzed aggregate rating distribution.
* Calculated average votes received by restaurants.
* Created rating distribution visualizations.

### Key Finding

Many restaurants had unrated or low-rated records, while highly rated restaurants generally received more votes.

---

## Task 2: Cuisine Combination Analysis

* Identified the most common cuisine combinations.
* Analyzed ratings of cuisine combinations.

### Key Finding

North Indian cuisine combinations appeared most frequently.

---

## Task 3: Geographic Analysis

* Visualized restaurant locations using latitude and longitude.
* Identified restaurant density clusters.

### Key Finding

Restaurant density was highest in New Delhi and surrounding regions.

---

## Task 4: Restaurant Chains Analysis

* Identified restaurant chains present in the dataset.
* Compared ratings and popularity of chains.

### Key Finding

Cafe Coffee Day was identified as the largest restaurant chain in the dataset.

---

# Major Insights

* North Indian cuisine dominated the restaurant dataset.
* New Delhi contained the highest concentration of restaurants.
* Price Range 1 restaurants were most common.
* Online delivery services were associated with higher ratings.
* Cafe Coffee Day was the largest restaurant chain.
* Geographic analysis revealed strong restaurant clustering in major cities.
